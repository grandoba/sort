# sort
This repository aims to cover all possible ways to sort. 


## How To Run 

**Sort in Python3**
```sh
#bash
cd src
# check how to run sort.py
python3 sort.py -h
# example 1: python3 sort.py 0_a
# example 2: python3 sort.py 4
# example 3: python3 sort.py 7
```
